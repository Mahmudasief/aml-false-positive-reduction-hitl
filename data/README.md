## Dataset

This project uses the Kaggle Credit Card Fraud Detection dataset
(European cardholders, September 2013).

- Source: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
- File: creditcard.csv
- Rows: ~284,807
- Label: Class (0 = normal, 1 = fraud)

Due to GitHub file size limits, the raw CSV is not stored in this repository.

### How to obtain the data
1. Download the dataset from Kaggle.
2. Extract `creditcard.csv`.
3. Place it in this folder: data/creditcard.csv
